# LFMAI Expo App
This is a React Native app built with [Expo](https://expo.dev).

## Features
- AI chat application powered by local on-device LLM (Large Language Model) models from Leap AI.

## Installation
1. Ensure you have Node.js and npm installed.
2. Clone the repository.
3. Navigate to the project directory and run `npm install`.

## Running the App
- To start the development server, use `npx expo start`.

## Building for Android
1. Run `npx expo run:android` to create a production build.
2. Follow the instructions in the terminal to complete the build process.
